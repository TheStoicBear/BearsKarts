# BearsKarts
A Simple GoKart racing script with a Race Track &amp; Karts


# Installation

---------------

1) Unpack ZIP File.
2) Drag and Drop [BearsKarts] into your Server/Resource Folder.
3) Navigate to our Server.cfg and add the following ensure [BearsKarts]
